# 📊 Evolution of Big Data & Cloud

---

# 📊 1. Evolution of Big Data (Why, What, How)

## ❓ Why Big Data emerged?
- Rapid growth of data from websites, apps, and devices
- Traditional databases couldn't handle large-scale data
- Need for better decision-making using data

## ❓ What is Big Data?
Big Data refers to very large and complex datasets.

### 3V’s:
- Volume → Huge amount of data
- Velocity → Fast data generation
- Variety → Different data types

## ❓ How Big Data is handled?
- Distributed systems (multiple machines)
- Tools:
  - Hadoop
  - Spark
- Parallel processing for speed

---

# ☁️ 2. Evolution of Cloud (Why, What, How)

## ❓ Why Cloud emerged?
- High cost of physical servers
- Need for scalable infrastructure
- Demand for global access

## ❓ What is Cloud Computing?
Cloud provides computing services over the internet.

## ❓ How Cloud works?
- Providers: AWS, GCP, Azure
- Pay-as-you-use model
- Easy scaling of resources

### Types:
- IaaS → Infrastructure
- PaaS → Platform
- SaaS → Software

---

# 🚀 Key Takeaway

- Big Data solves large-scale data problems
- Cloud provides scalable infrastructure to handle it
